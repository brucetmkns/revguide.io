Everything your team needs to execute

Three powerful tools that work together to surface the right information at the right time.

Contextual Banners

Display alerts, warnings, and guidance on HubSpot records based on property conditions. Trigger the right message when it matters most.

  • Property-based rule engine with AND/OR logic
  • 5 banner types: Info, Success, Warning, Error, Embed
  • Embed videos from YouTube, Loom, Vimeo
  • Link banners to related plays

Sales Plays

Create structured playbooks for competitors, objection handling, processes, and best practices. Always accessible from the Chrome sidebar.

  • Multi-section playbooks with drag-to-reorder
  • Competitor battle cards and objection handlers
  • Conditional display based on record properties
  • Persistent access via Chrome Side Panel

Field Glossary

Build a knowledge base with definitions that appear as tooltips right next to HubSpot property labels. Reduce confusion, improve data quality.

  • Tooltips on HubSpot property labels
  • Import fields directly from HubSpot API
  • Hierarchical organization with categories
  • Optimized for 1000+ entries
