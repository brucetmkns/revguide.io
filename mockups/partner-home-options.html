Summary

Key changes across all options:

1. Simplified sidebar - Remove Banners, Plays, Wiki (partners don't create content directly)
2. 3-step onboarding - Install Extension, Connect Client, Deploy Content
3. Client-focused - Show clients prominently since that's the partner's primary job
4. Libraries emphasis - Partners deploy pre-built content, not create from scratch

Recommendation: Option C provides the most visual consistency with the existing client home, which reduces cognitive load for users who transition between being a client and a partner. Consider adding the pending requests alert from Option B.
