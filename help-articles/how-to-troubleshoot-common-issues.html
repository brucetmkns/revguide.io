How to Troubleshoot Common Issues

Quick fixes for the most common RevGuide problems

This guide covers the most frequently encountered issues and their solutions. If you can't find your issue here, contact support for additional help.

Extension Issues

Extension Not Working in HubSpot

Symptoms: No RevGuide icon, no tooltips, no banners appearing

Solutions:
  1. Verify the extension is installed: Check Chrome's extension menu (puzzle icon)
  2. Ensure it's enabled: Extensions > RevGuide > toggle ON
  3. Check you're on a HubSpot page (app.hubspot.com)
  4. Refresh the HubSpot page
  5. If still not working, remove and reinstall the extension
Content Not Syncing

Symptoms: Changes in admin panel don't appear in HubSpot

Solutions:
  1. Wait a few seconds - sync can take a moment
  2. Refresh the HubSpot page
  3. Check that the content is enabled (not disabled)
  4. Verify you're signed in to the same organization
  5. Clear browser cache and refresh

Banner Issues

Banner Not Appearing

Symptoms: Created a banner but it doesn't show in HubSpot

Solutions:
  1. Check enabled status: Ensure the banner is toggled ON
  2. Verify URL pattern: Confirm the pattern matches the current page
    • Try using * to match all pages as a test
    • Check for typos in URL patterns
  3. Check display settings: Verify "Show Banners" is enabled in Settings
  4. Refresh the page: Sometimes content needs a page refresh
Banner Appearing on Wrong Pages

Symptoms: Banner shows on pages where it shouldn't

Solutions:
  1. Review your URL pattern - it may be too broad
  2. Change from * to more specific patterns like /contacts/*
  3. Add multiple specific patterns instead of one broad pattern
  4. Test by navigating to different page types

Wiki/Tooltip Issues

Tooltips Not Appearing

Symptoms: Wiki entries exist but no tooltips show in HubSpot

Solutions:
  1. Check trigger word: Ensure the entry has a trigger word set (not just a title)
  2. Verify enabled: Entry must be enabled
  3. Check match type: "Exact" only matches complete words
  4. Look at the page: The trigger word must actually appear on the page
  5. Check page type filter: Entry may be limited to specific page types
  6. Verify case: Triggers are case-insensitive but check spelling
Wrong Tooltip Showing

Symptoms: A different definition appears than expected

Solutions:
  1. Check for duplicates: Search for the trigger word in Wiki to find conflicts
  2. Review aliases: An alias on another entry may be matching
  3. Check priority: Higher priority entries take precedence
  4. Resolve conflicts: Adjust or merge duplicate entries

HubSpot Connection Issues

Can't Connect to HubSpot

Symptoms: OAuth flow fails or connection doesn't complete

Solutions:
  1. Ensure you have admin/super admin access in HubSpot
  2. Disable popup blockers during OAuth
  3. Clear browser cookies for both RevGuide and HubSpot
  4. Try in an incognito/private window
  5. Check that third-party cookies are enabled
Field Import Not Working

Symptoms: Can't import fields from HubSpot or fields list is empty

Solutions:
  1. Verify HubSpot is connected (check Settings)
  2. If disconnected, reconnect HubSpot
  3. Ensure you have permission to view properties in HubSpot
  4. Try selecting a different object type
  5. Refresh the page and try again

Login/Authentication Issues

Can't Sign In

Symptoms: Login fails or page redirects repeatedly

Solutions:
  1. Verify email and password are correct
  2. Check for caps lock
  3. Try "Forgot Password" to reset
  4. Clear browser cache and cookies for RevGuide
  5. Try a different browser
  6. Disable browser extensions that might interfere
Session Expired Frequently

Symptoms: Getting logged out unexpectedly

Solutions:
  1. Check that cookies aren't being blocked
  2. Disable "clear cookies on exit" browser setting
  3. Use the web app instead of extension if available
  4. Check for conflicting browser extensions

Permission Issues

Can't Edit Content

Symptoms: Edit buttons missing or disabled

Solutions:
  1. Check your role - Viewers can't edit content
  2. Look for "View Only" badge indicating restricted access
  3. Ask an admin to upgrade your role to Editor or Admin
Can't Access Settings

Symptoms: Settings options hidden or greyed out

Solutions:
  1. Some settings require Admin or Owner role
  2. HubSpot connection requires Admin access
  3. Team management requires Admin access
  4. Contact your organization admin for role changes

Performance Issues

Slow Loading

Symptoms: Pages take a long time to load

Solutions:
  1. Check your internet connection
  2. Clear browser cache
  3. Disable other browser extensions temporarily
  4. Try in incognito mode
  5. If using many wiki entries, consider disabling some

General Troubleshooting Steps

When something isn't working, try these steps in order:

  1. Refresh the page - Solves many temporary issues
  2. Check enabled status - Ensure content is toggled on
  3. Verify your role - Some features require higher permissions
  4. Clear browser cache - Removes stale data
  5. Sign out and back in - Refreshes your session
  6. Try incognito mode - Rules out extension conflicts
  7. Reinstall extension - Last resort for extension issues
Tip: Before contacting support, try to reproduce the issue in an incognito window. This helps identify if browser extensions or settings are causing the problem.

Getting Help

If these solutions don't resolve your issue:

  1. Note the exact error message (if any)
  2. Take a screenshot of the problem
  3. Note what steps led to the issue
  4. Contact your organization admin first
  5. If needed, reach out to RevGuide support with details
Note: When reporting issues, include your browser version, whether you're using the extension or web app, and any error messages you see.
