How to Manage Display Settings

Configure how RevGuide content appears in HubSpot

Display settings control how banners, plays, and other RevGuide features appear when you're working in HubSpot. Customize these settings to match your team's preferences.

Accessing Display Settings

1 Navigate to Settings

Open the RevGuide admin panel and click "Settings" in the left sidebar.

2 Find Display Options

Scroll to the "Display Options" card. These settings affect how content appears in HubSpot.

Available Settings

Show Banners

Setting Effect
ON (Default) Banners appear in HubSpot according to their conditions
OFF No banners appear, even if enabled

Use this to temporarily hide all banners without disabling individual banners.

Show Battle Cards (Plays)

Setting Effect
ON (Default) Plays are available in the side panel
OFF Plays tab is hidden in the side panel

Show Presentations (Media)

Setting Effect
ON (Default) Media content is available in the side panel
OFF Media tab is hidden in the side panel

Show Admin Links

Setting Effect
ON (Default) Quick link to admin panel appears in side panel
OFF Admin link is hidden (users go to web app directly)

Banner Position

Position Where Banners Appear
Top (Default) Above the main HubSpot content area
Bottom Below the main HubSpot content area
Tip: Top position is more visible for important announcements. Bottom position is less intrusive for informational content.

Changing Settings

1 Toggle the Setting

Click the toggle switch next to the setting you want to change.

2 Auto-Save

Settings save automatically when changed. You'll see a confirmation toast.

3 Refresh HubSpot

Changes take effect immediately. Refresh HubSpot pages to see the changes.

Organization-Wide vs. Personal Settings

Important: Display settings are organization-wide. When you change a setting, it affects all users in your organization.

Settings apply to:

  • All team members in your organization
  • Both extension and web app users
  • All content of the affected type

Common Use Cases

Launching RevGuide Gradually

Start with minimal features and enable more over time:

  1. Disable all display options initially
  2. Create and test your content
  3. Enable wiki tooltips first (most unobtrusive)
  4. Add plays once wiki is established
  5. Enable banners for major announcements

Reducing Distractions

If users report too much content:

  • Move banners to bottom position
  • Disable banners temporarily
  • Review and reduce active content
  • Make banner conditions more specific

Focus on Specific Features

Emphasize what matters most:

  • Sales team: Enable plays, keep banners minimal
  • Onboarding: Enable all features including media
  • Announcements: Enable banners at top position

Settings for Different Roles

Role Can View Settings Can Change Settings
Viewer Limited No
Editor Yes No
Admin Yes Yes
Owner Yes Yes

Best Practices

Communicate Changes

  • Let team members know when you change display settings
  • Explain why settings were changed
  • Create a banner to announce major changes

Test Before Rolling Out

  • Test setting changes during off-peak hours
  • Get feedback from a few users first
  • Be ready to revert if issues arise

Document Your Configuration

  • Note your preferred settings configuration
  • Include settings documentation in team onboarding
  • Review settings when team needs change

Troubleshooting

Setting Not Taking Effect

  • Refresh the HubSpot page
  • Clear browser cache
  • Verify the setting saved (check the toggle state)
  • Sign out and back in

Can't Change Settings

  • Verify you have Admin or Owner role
  • Check that you're on the Settings page (not a content page)
  • Try refreshing the admin panel

Related Settings

Other areas that affect display:

  • Banner conditions: Individual banner URL patterns
  • Wiki rules: Tooltip frequency, match type, page type filters
  • Content enabled status: Individual item on/off toggles
