How to Embed Media in Banners

Add videos, images, and rich content to your banners

Enhance your banners with embedded media to create more engaging and informative content. RevGuide supports videos from popular platforms, images, and rich formatting.

Supported Media Types

Type Supported Sources
Videos Loom, YouTube, Vimeo
Images Any publicly accessible URL
Links Any URL
Rich Text Bold, italic, lists, headings

Embedding Loom Videos

Loom is ideal for quick training videos and process walkthroughs.

1 Get Your Loom Share Link

Open your Loom video and click "Share". Copy the share link (e.g., https://www.loom.com/share/abc123...)

2 Add to Banner

In the banner editor, paste the Loom URL directly into the message field. RevGuide will automatically convert it to an embedded video.

3 Preview

Save the banner and preview it in HubSpot. The video will appear as an expandable player.

Tip: Keep Loom videos short (under 2 minutes) for banners. Link to longer content instead of embedding it.

Embedding YouTube Videos

1 Get the YouTube URL

Copy the video URL from your browser or click "Share" on YouTube and copy the link.

2 Paste in Banner

Add the YouTube URL to your banner message. Supported formats:

  • https://www.youtube.com/watch?v=VIDEO_ID
  • https://youtu.be/VIDEO_ID

Embedding Vimeo Videos

1 Get the Vimeo URL

Copy the video URL (e.g., https://vimeo.com/123456789)

2 Add to Banner

Paste the Vimeo URL into your banner message.

Adding Images

1 Host Your Image

Upload your image to a hosting service and get the direct image URL. The URL should end in .jpg, .png, .gif, or .webp.

2 Add to Banner

Use the image button in the rich text editor, or paste the image URL directly.

Note: Use images hosted on reliable services. Images hosted on temporary URLs may break over time.

Adding Links

Using the Rich Text Editor

  1. Select the text you want to link
  2. Click the link button in the toolbar
  3. Enter the URL
  4. Click to save

Link Best Practices

  • Use descriptive link text (not "click here")
  • Links open in new tabs by default
  • Test links to ensure they work

Rich Text Formatting

The banner editor supports standard formatting:

Format How to Use
Bold Select text, click B or Ctrl/Cmd+B
Italic Select text, click I or Ctrl/Cmd+I
Bullet List Click bullet list button
Numbered List Click numbered list button
Heading Select heading size from dropdown

Design Tips for Media Banners

Keep It Scannable

  • Use short paragraphs
  • Break up text with lists
  • One main message per banner

Video Recommendations

  • Keep banner videos under 2 minutes
  • Front-load the key information
  • Include a text summary alongside video
  • Consider using a link for longer videos

Image Guidelines

  • Use appropriate image sizes (not too large)
  • Ensure good contrast and readability
  • Add alt text for accessibility
  • Optimize file sizes for fast loading

Troubleshooting

Video Not Loading

  • Check that the video URL is correct and public
  • Ensure the video isn't password protected
  • Try using the embed URL instead of the share URL

Image Not Displaying

  • Verify the image URL is publicly accessible
  • Check that the URL points directly to the image file
  • Try a different hosting service
